
# ลอยกระทงออนไลน์ – GitHub Pages Ready

เว็บไซต์กิจกรรมลอยกระทงออนไลน์แบบหน้าเดียว (Static) พร้อมอัปขึ้น GitHub Pages

## วิธีใช้งานด่วน
1) อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น GitHub Repository ใหม่
2) ไปที่ **Settings → Pages**
   - Source: เลือก `main` branch
   - Folder: `/ (root)`
3) กด Save แล้วรอระบบเผยแพร่
4) เปิดลิงก์: `https://<your-username>.github.io/<repo-name>/`

## เปลี่ยนโลโก้
แทนที่ไฟล์ `assets/logo.png` ด้วยโลโก้หน่วยงานของคุณ (แนะนำ PNG พื้นใส)
